Summary: Latitude vs. X

The purpose of this project was to analyze how societal factors effect the spread of COVID-19 in the United States of America. To accomplish this analysis, I pulled data from the Johns Hopkins Covid-19 github repository and matched that data to demographic data from the US Census API on a county by county basis.

After Assembling the dataset, I used Matplotlib to create scatter plots of the number of confirmed cases on the 50th day since the first case appeared in a county vs. poverty rate, population density, median household income, and hospital quality. I then created heatmaps of the COVID-19 confirmed cases and deaths to better visualize the problem. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
